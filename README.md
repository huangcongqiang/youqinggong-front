# youqinggong-front

有轻工前台应用仓库，对应平台的官网首页、企业端和人才端。

## 功能范围

- 官网首页
- 企业端：入驻、发布任务、人才广场、人才详情、项目沟通、协作空间、验收评分
- 人才端：入驻、任务广场、项目沟通、协作空间、验收评分
- H5 / PC 自适应

## 技术栈

- Vue 3
- Vue Router
- Vite
- CSS 变量 + 响应式布局

## 本地运行

```bash
npm install
npm run dev
```

默认地址：

- `http://localhost:5173`

## 目录

- `src/`：页面、组件、路由与前台 API 封装
- `docs/`：项目文档副本
- `deploy/`：部署模板副本
