# 主线 10 / 11 UI 评分总表

> 说明：本表记录 PC 前台、H5 前台与 Admin 当前 UI 基线分数、问题级别和整改状态。  
> 当前版本已完成主线 10 的阶段一基线建档，并开始登记主线 11 的后台截图建档范围。

## 评分标准

- 总分：`10`
- 维度：
  - 视觉层级
  - 信息密度
  - 端适配正确性
  - 操作效率
  - 一致性
- 问题级别：
  - `P0`：影响可用性、识别或主操作完成
  - `P1`：明显影响专业感、效率、阅读流或布局稳定性
  - `P2`：细节粗糙、风格不统一、局部视觉噪声

## 跨页共性问题

### P0

- 当前对象工作页首屏阻断问题已基本清零，后续进入 `P1` 收口

### P1

- PC/H5 壳层抬头和页面级标题的叠加已减少，但入口页仍有少量次级信息抢主操作注意力
- 门户与工作台首屏文案明显减量，但平台方式 / 案例 / 解释句仍有继续压缩空间
- 玻璃感和发光感偏重，稳重、可信和高端感不够统一
- 工作页头部、筛选芯片和动作带虽然已开始统一，但业务页之间仍有各自造一套首屏骨架的倾向
- 第一批高频对象页里，聊天、协作 / 验收和首批业务页的单页最终过门判断已经完成；第一批高频入口页里，H5 落地页、H5 注册页、H5 登录页、PC 官网首页、PC 注册页、PC 企业工作台、PC 人才工作台、H5 企业工作台与 H5 人才工作台当前都已过门；第二批与第三批低分页、对象详情、任务池和入驻页也已完成收口，当前正式转入阶段五整端复拍

### P2

- 颜色、半径、边框和按钮风格在两端仍有轻微漂移
- 列表页、对象页、表单页的标题区没有完全统一
- 多处空态仍偏“说明页”表达

## 当前优先级顺序

1. 阶段五终轮总复审与阻断项清零
2. 整端重新核对高频页 `>= 8`、其它页 `>= 7`

4. 达标后收口主线 10，并自动切主线 11
