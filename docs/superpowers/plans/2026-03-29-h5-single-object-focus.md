# H5 单对象处理页首屏收口

## 阶段目标

- 把 H5 对象页统一收成“当前对象 + 当前状态 + 一个主动作”的单焦点首屏
- 聊天详情页去掉重复摘要，任务确认卡不再首屏展开版本历史
- 记录详情首屏继续压缩状态区，避免回到 dashboard 式对象页

## 实际结果

- `MessagesPage.vue` 已移除重复的 `SummarySection`，聊天详情页当前不再重复承接“会话详情 + 当前会话 + 聊天头部”三层摘要
- `MessagesPage.vue` 里的任务确认卡当前只保留预算、工期、协作安排、档期和主动作，版本历史与更长说明已下沉到任务详情 / 任务确认 sheet
- `RecordDetailPage.vue` 的状态卡已继续压缩，首屏更接近对象摘要而不是 dashboard 指标面板

## 验证

- `frontend_mobile_h5`：`npm run build` 通过
- H5 `MessagesPage` 构建产物进一步降到约 `768.61 kB`
