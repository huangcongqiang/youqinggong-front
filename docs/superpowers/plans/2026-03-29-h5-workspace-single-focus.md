# H5 协作页单焦点化

## 阶段目标

把 H5 协作页从“压缩版桌面工作区”收成真正的移动端执行台。

## 本阶段范围

- 首屏只保留当前任务摘要
- 首屏只保留一个主动作和一个次动作
- 切换任务、任务详情、最近记录改成轻入口
- 当前节点只露一个，完整节点与重表单下沉
- 去掉重复的 sticky 动作带

## 当前结果

- 任务切换已改为轻入口 + sheet，不再整列铺在首屏
- 首屏当前固定为“当前任务 + 主次动作 + 轻入口 + 当前节点”
- 完整节点、最近记录、任务详情继续通过详情层承接
- 底部重复 sticky 动作带已移除

## 下一步

默认直接进入第三阶段：记录 / 验收详情链 App 化。
